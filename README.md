# ~*~ Retro 2000s Website ~*~

Un sitio web funcional con estética retro de los 2000s, creado con HTML/CSS/JS puro.

## Contenido

| Sección | Descripción |
|---------|-------------|
| **Inicio** | Portada con contador de visitas, marquee, update log y badges 88x31 |
| **Música** | Top canciones 2000-2007, géneros que marcaron la década |
| **Tecnología** | Nokia 3310, Windows XP, dial-up, cibercafés, gadgets icónicos |
| **Videojuegos** | PS2, Xbox, GameCube, GBA, CS 1.6, Ragnarök, Gunbound, MapleStory |
| **Internet** | MSN Messenger, GeoCities, Fotolog, Napster, Ares, NeoPets |
| **Guestbook** | Libro de firmas con emoticones clásicos (localStorage) |
| **Galería** | 30+ GIFs animados con CSS (sparkles, under construction, emoticones) |
| **Quiz** | 10 preguntas para medir qué tan 2000s eres |

## Funcionalidades

- **Fondo estelar animado** con canvas + estrellas titilantes
- **Cursor con estela de partículas** de colores neón
- **Contador de visitas** con hitos festejados al llegar a números clave
- **Reproductor estilo Winamp** con melodías 8-bit (Web Audio API)
- **Guestbook interactivo** con almacenamiento local
- **Quiz interactivo** con ranking y resultados
- **Scrollbar personalizada** retro neón
- **Compatible con** IE 5.0+, Netscape Navigator, Firefox 1.0 (mentira, pero se ve cool decirlo)

## Cómo correrlo

```bash
python3 -m http.server 8080
# o
npx serve .
```

Abrir `http://localhost:8080` en el navegador.

## Stack

- HTML 4.01 (estricto mentalmente)
- CSS3 con animaciones y fuentes de Google (Press Start 2P)
- JavaScript vanilla (canvas, localStorage, Web Audio API)
- Sin frameworks, sin build tools — como en los 2000s

## Créditos

Hecho con Notepad, inspirado en GeoCities, Angelfire y los sitios de la era dorada del internet.
